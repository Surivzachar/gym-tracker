## 🎯 TRAINING TECHNIQUES

### SUPERSET 1A/1B
```
Exercise A → Exercise B → Rest 60-90s → Repeat
```
Example: Incline Press → Pull-ups → Rest → Repeat

### GIANT SET 1A/1B/1C
```
Exercise A → Exercise B → Exercise C → Rest 90-120s → Repeat
```
Example: Incline Curls → Barbell Curls → Hammer Curls → Rest

### DROP SET (last set)
```
Normal set → Drop weight 20-30% → Go to failure
```
Example: Leg Extensions 30kg x 12 → 20kg x 15

### CIRCUIT (Round 1,2,3,4)
```
Do all exercises in order → Rest 60-90s → Repeat
```

## 📈 PROGRESSIVE OVERLOAD

### When Reps are SAME (e.g., 4x12,12,12,12)
```
✅ ALL 4 SETS = SAME WEIGHT

Week 1: 20kg x 12,12,12,12 ✓
Week 2: 22.5kg (add weight)
```

### When Reps DECREASE (e.g., 4x12,12,10,10)
```
⚡ INCREASE WEIGHT as reps decrease

Week 1:
Set 1-2: 20kg x 12 reps (lighter)
Set 3-4: 22.5kg x 10 reps (heavier)

Week 2 (if hit all targets):
Set 1-2: 22.5kg x 12 reps
Set 3-4: 25kg x 10 reps
```

### The Rule:
```
Hit ALL targets? → ADD WEIGHT ✅
Missed targets? → KEEP SAME ⏸️
Too heavy? → Go back, add REPS ↩️
```

### How Much to Add:
```
Upper body: +2.5kg
Lower body: +5kg
```

## 📋 YOUR WORKOUTS

### TUESDAY - SUPERSETS
```
1A: Incline DB Press (12,12,10,10)
   Sets 1-2: Lighter weight
   Sets 3-4: Heavier weight (+2.5kg)

1B: Pull-ups (12,10,10,8)
   Weight increases each set OR
   Reps decrease if bodyweight

2A: Flat DB Press (15,12,12)
   Set 1: Lighter
   Sets 2-3: Heavier

2B: Cable Rows (15,12,12)
   Set 1: Lighter
   Sets 2-3: Heavier

3A+3B: Same pattern
```

### WEDNESDAY - GIANT SETS
```
GIANT SET 1 (Biceps):
Incline Curls → Barbell Curls → Hammer Curls
Same weight all 3 sets each exercise

GIANT SET 2 (Triceps):
Overhead Ext → Rope Pushdowns → Close Grip
Same weight all 3 sets each
Last set Rope Pushdowns = DROP SET!
```

### THURSDAY - STRAIGHT SETS
```
Most exercises = same weight all sets
(Unless reps decrease)

Quads: 15,12,12,10 = increase weight
Hamstrings: Usually same weight all sets
Calves: 20,18,15,12 = increase weight
```

### FRIDAY - STRAIGHT SETS
```
Press: 12,12,10,10 = increase weight
Lateral Raises: 15,15,12,12 = same then heavier
All raises: Usually 3-4 sets same weight
```

### SUNDAY - CIRCUIT
```
All 8 exercises x 4 rounds
Same weight all 4 rounds
Focus on speed and form
```

## 🎯 AT THE GYM

**Before starting:**
- Check last week's weights in app
- Know the technique (superset/giant set/circuit)

**During workout:**
- Log weight + reps immediately after each set
- Rest 60-90s (supersets), 90-120s (giant sets)
- Focus on form first, weight second

**After workout:**
- Did I hit all target reps? Yes/No
- If YES → add weight next week
- If NO → keep same weight, try again

## 📊 DURING FAT LOSS (12 WEEKS)

**Weeks 1-4:** Will get stronger ✅
**Weeks 5-8:** Strength plateaus (normal) ⏸️
**Weeks 9-12:** Slight decrease OK (very lean) ✓

**Goal:** MAINTAIN strength = SUCCESS
**Not goal:** Big strength gains (you're in deficit)

---

## ✅ REMEMBER

✔ Progressive reps scheme = increase weight as reps decrease
✔ Same reps all sets = same weight all sets (until you progress)
✔ Hit all targets = add weight next week
✔ Form > Weight always
✔ Track everything in app
✔ Rest 60-120 seconds between sets
✔ During cut, maintaining strength = winning
